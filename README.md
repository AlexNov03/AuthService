# Репозиторий бэкенда для добавления, получения и изменения списка задач пользователей

Запуск проекта 
```shell
make start
```
Остановка всех контейнеров проекта
```shell
make stop
```
## Работа с миграциями:
Применить миграцию
```shell
make migrate-up
```
Откатить миграцию
```shell
make migrate-down
```
## Используемые технологии:
- GO
- PostgreSQL
- Docker
- Grpc/protobuf
